HOW TO RUN:
    - Navigate to one folder above the Controller folder
    - Run the command: java Controller.Window

INSTRUCTIONS & FUNCTIONALITY:

    BASIC CONTROLS:
        - Left click selects and moves the chess pieces
        - Right click un-selects a chess piece

    VISUAL:
        - F1 toggles transparency
        - F2 toggles the marker
        - F3 toggles the ability to view the moves a piece can go to
        - F4 toggles the ability to view the moves all the pieces can go to

    SAVING/LOADING:
        - S saves the current state of the board
        - L loads the current state of the board
